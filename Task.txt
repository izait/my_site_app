Веб интерфейс управлением микро-crm

1. Использовать React js
2. 3 страницы : города, улицы, пользователи
3. В верхнем правом углу имя авторизованного пользователя
4. Использовать базовый css(без плагинов и бутстрапов)
5. Для асинхронных запросов использовать только fetch и Promise
6. Использовать Lambda выражения(лямбда функции)
7. Разделять проект на компоненты

8. Сделать страницу для авторизации + Реализовать авторизацию пользователей
9.Доступ к CRM только пользователям с ролью ROLE_ADMIN
10. Пользователям с другими ролями выводить ошибку !модальным окном! с текстом "Нет доступа"
11. В правом верхнем углу отображать логин авторизованного пользователя и кнопку для разлогинивания
